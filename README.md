### Level Calculation <a name="level"></a>

- No Max Level
- Score required to level up

$Level = \sqrt{Score} * 0.2$

```
e.g. If user has 3600 Score, their level is 12,

3600 = (Level / 0.2) ^ 2
âˆš3600 = Level / 0.2
Level = 60 * 0.2
Level = 12
```
